# e-plantShopping

e-plantShopping is a React-based plant shopping application that allows users to browse a collection of plants, add plants to a shopping cart, and manage their selections through an interactive user interface.

## Features

* Browse a variety of plants
* View plant categories and details
* Add plants to the shopping cart
* Manage cart items and quantities
* Responsive React user interface

## Technologies Used

* React
* JavaScript
* HTML
* CSS
* Vite

This project was developed as part of the IBM Front-End Development learning program.
